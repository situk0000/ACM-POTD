
## 🔗 Problem:
Remove Duplicates from Sorted Array

## 🧠 Approach:
- I used the two-pointer approach
- One pointer (i) keeps track of the last unique element
- The other pointer (j) traverses the array
- Whenever a new element is found, it is placed at the next position
